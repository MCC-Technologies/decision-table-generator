# Decision Table Generator

A simple utility for generating structured decision tables for testing, analysis, and validation scenarios.

## 🚀 Purpose

This tool helps developers, QA engineers, and analysts create decision tables that map conditions to outcomes. It is especially useful for:

* Test case generation
* Business rule validation
* Edge case identification
* Logic modeling

## 🧰 Features (Planned / In Progress)

* Generate decision tables from input conditions
* Export tables in readable formats
* Support for multiple rule combinations
* Lightweight and easy to use

## 🛠️ Usage

> *Documentation coming soon.*

Once implemented, this tool will allow users to define conditions and automatically generate all possible decision combinations.
